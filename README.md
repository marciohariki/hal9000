# Hal9000 - AngularJS and Django REST

##Requirements

- Python 2.7.10
- Django 1.7.1
- Npm v2.1.18
- Bower 1.5.2

##Instalation

```
npm install
python manage.py syncdb

```

##Running

```
python manage.py runserver

```

##Extra Notes

If you want to install any bower package go to static folder
